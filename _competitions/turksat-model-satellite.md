# {{ page.title }}

I compete in <a href="https://teknofest.org/en/competitions/competition/28" target="_blank">Teknofest Türksat Model Satellite (CANSAT) Competition</a> for 2 years with a team ([Bursat Model Satellite Team](https://github.com/btu-bursat)) in my university.

I was responsible for designing and developing embedded software for the model satellite we were producing. I also designed team logo, choose electrical components and were software team leader.

Topics I learned during this competition:

* Microcontroller software
  * State machines
* Microcontroller interfaces
  * UART
  * I2C
  * SPI
* STM32
  * HAL drivers
* Choosing embedded device parts
  * Microcontrollers
  * Sensors
  * Communication devices
* Python
  * Thread
* Logo design
  * Inkspace
